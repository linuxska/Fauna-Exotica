********  Cambios realizados (que no son pocos)

* Base de datos modificada (Recuperar contraseña):
- La tabla 'usuarios' contiene un campo más, password_recuperacion. Éste sirve para lo que suena. Si un usuario solicita
recuperar su clave, se le crea una nueva aleatoria, y se añade en dicho campo. Si el usuario introduce esa clave, pasará a
ser su clave principal (borando la de recuperación). Esto se hizo así porque no se pueden recuperar las claves codificadas
en la BD, el md5 no se puede descodificar.


* Para poder enviar emails (Recuperar contraseña y contactar):
- Creamos la cuenta lauscar.sl@gmail.com. El servidor se conecta a la cuenta por SMTP y envia el correo desde la misma.
Para que esto funcione configuramos php.ini habilitando la extensión  php_openssl.dll
NOTA: reiniciar servidor para ver cambios

* Eliminamos index.php de las url ( /inicio en lugar de index.php/inicio )
- Creando un archivo .htaccess para cambiar el rutamiento que hace Apache. Para que funcione ir a httpd.conf y habilitar 
el módulo mod_rewrite.so
También se modificó el config de Codeigniter...
NOTA: reiniciar servidor para ver cambios

* Métodos privados
- Todo aquel método que no fuese llamado por el cliente sino internamente se pasó a 'private function' para más seguridad.
